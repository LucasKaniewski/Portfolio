Site Marchand :

Mon Bînome : Delannoy Ewan

Dans le cadre de notre premiere année de bts , nous avons du effectuer en binome un site marchand en un délai de 2 mois , le but étant de nous faire manipuler une bases de donnée utiliser le language php et apprendre à travailler en collaboration

Le rojet devait représenter un site marchand permettant la vente de produit

Les objectifs :
- Un système de création du compte,ainsi qu'une connexion à leur compte
- L'ajout d'un produit dans leur panier
- Gérer les produits dans (Suprression etc...)
- La création d'un back office (Visonnage des paniers des clients,des comptes etc.)
- L'affichage des produits dans les catalogues depuis la bdd

Les langages informatiques utilisés :
- Html
- Css
- Php
- Sql
